


# 1. 2的n次方

```c++
2^8 = 256
2^10 = 1024
2^12 = 4096
2^16 = 65536
2^20 = 1048576
2^32 = 4294967296
```

# 2. 整型取值范围

```cpp
char(1B): signed -128~127
		  unsigned 0~255
		  
short(2B): signed -32768~32767
		   unsigned 0~65535
		   
int(4B): signed -2147483648~2147483647,
		 unsigned 0~4294967295
```

# 3. 进制 & 全1规律

```c++
1 Byte = 8 bit
1位十六进制 = 4位二进制
8位全1 = 0xFF = 255
16位全1 = 0xFFFF = 65535
n位全1 = 2^n - 1
```

# 4. 位运算秒杀结论

```c++
x & (x-1)：消除最右侧1个1，统计1的个数专用
x & 1：取最低位，判断奇偶
x << 1：等价 ×2
x >> 1：等价 ÷2
~x：按位取反，有符号数下 ~x = -x -1
a ^ a = 0, a ^ 0 = a
```

# 5. ASCII 必背

```c++
'0' = 48
'A' = 65
'a' = 97
大小写差值：32
```
# 6. 补码核心
正数：`原码=反码=补码`
负数补码：`数值位取反 + 1`
有符号数：`负数最小值绝对值 = 正数最大值 + 1`
