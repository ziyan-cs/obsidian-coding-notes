# 1. 时间复杂度

	- `sort` → $O(N·log N)$

	- `unordered` → $O(1)$

# 2. 空间复杂度

	- 平衡 → $O(log N)$
	- 极端 → $O(N)$
- 原地算法 → $O(1)$

# 3. Master 定理

### 3.1 公式

- 前提：分治型递归的子问题规模相同
- 递归式：$T(N)=aT(N/b​)+O(N^d)$
	 - $a$：递归调用次数（a≥1）
	 - $N/b$：每次递归的子问题数据规模（b>1）
	 - $O(N^d)$：递归外操作的时间复杂度

### 3.2 三种情况

- $log(b,​a) > d$    →    $T(n)=O(N^(log(b , ​a)))$
- $log(b,​a) = d$    →    $T(n)=O(N^d·logN)$
- $log(b,​a) < d$    →    $T(n)=O(N^d)$

### 3.3 典型应用

- 递归求数组最大值：
	- $T(N)=1⋅T(N/2)+O(1)$
	- $a = 1$ , $b = 2$ , $d = 0$
	- $log(2,1) = 0 = d$   → 情况 2
	- 复杂度：$O(logN)$

- 归并排序：
	- $T(N)=2⋅T(N/2)+O(1)$
	- $a = 2$ , $b = 2$ , $d = 0$
	- $log(2,1) = 0 = d$   → 情况 2
	- 复杂度：$O(N·logN)$

- 二分查找：
	- $T(N)=1⋅T(N/2)+O(1)$
	- $a = 1$ , $b = 2$ , $d = 0$
	- $log(2,1) = 0 = d$   → 情况 2
	- 复杂度：$O(logN)$

# 5. 常见算法复杂度总结

- 排序 ：
	- 冒泡 / 选择 / 插入 → $O(N^2)$
	- 归并 / 快排 → $O(N·log N)$

- 查找：
	- 二分查找 → $O(log N)$

- 数据结构：
	- 栈 / 队列 → $O(1)$
	- 堆 → $O(log N)$
	- 哈希 → $O(1)$

- 增长速度排序：
	- $O(1) < O(log N) < O(N) < O(N·log N) < O(N^2) < O(2^N)$
# 6. 复杂度分析套路

- 顺序：
	- 找循环  
	- 看嵌套层数  
	- 看是否折半  
	- 看是否递归  
	- 忽略常数
