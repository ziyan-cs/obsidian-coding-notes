# 0. 核心

- **一段连续内存中的同类型元素集合**
- - 连续存储，通过 **首地址 + 偏移量** 快速定位任意元素
- 相关特征：
    - 按下标快速访问 $O(1)$
    - 顺序遍历、统计、修改 $O(1)$
    - 数据规模已知、上界明确
- 优势：
    - 连续存储
    - 随机访问快
    - 常数小、实现简单

# 2. 常用操作与注意事项

- 数组作为函数参数：
	- 会退化为 **指针**，丢失长度信息，需额外传递长度
- 动态数组（vector，*C++* 推荐）：
	- 自动管理内存，支持动态扩容，可随机访问
- 注意：
	- **越界访问**
	- **局部数组未初始化**
	- **数组名是常量指针**
	- **二维数组行优先存储**
