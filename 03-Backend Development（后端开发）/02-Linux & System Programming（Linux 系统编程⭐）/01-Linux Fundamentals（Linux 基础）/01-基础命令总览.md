
# 0.核心

- `Shell` 是 **用户与操作系统内核交互的命令行解释器**
- `sudo [cmd]`：以 `root` 权限执行命令（*Ubuntu* 常用）
- 命令本质是操作四大核心对象：
    - 文件系统（目录 / 文件）
    - 进程
    - 权限
    - 输入输出流

# 1. 帮助命令

- `man [cmd/config]`：获得外部命令/配置文件的详细帮助手册
- `help [builtin_cmd]`：获得 `Shell` 内置命令的帮助信息
- `[cmd]--help`：快速查看命令参数概述
- `type []`： 判断命令类型（`external` / `builtin`）

# 2. 关机 & 重启

- `halt`：立即关机
- `reboot`：立即重启
- `sync`：同步内存数据到磁盘
- `poweroff`：立即关机

- `shutdown`：默认一分钟后关机
- `shutdown -h now`：立即关机
- `shutdown -h 1`：一分钟后关机（自定义时间）
- `shutdown -r now`：立即重启

# 3. 运行级别

- 传统运行级别：
	- `0`：关机
	- `1`：单用户模式【找回丢失密码】
	- `2`：多用户状态无网络服务
	- `3`：多用户状态有网络服务
	- `4`：系统未使用保留给用户
	- `5`：图形界面（默认）
	- `6`：系统重启
	- `init [级别]`：切换运行级别（例如  `init 3`）
	

- 现代 *Ubuntu* 等系统用 `systemd 目标（target）` 替代传统运行级别，`init` 命令仅做兼容：
	- `0` ↔ `poweroff.target`（关机）
	- `3` ↔ `multi-user.target`（多用户命令行，服务器常用）
	- `5` ↔ `graphical.target`（图形界面，桌面常用）
	- `6` ↔ `reboot.target`（重启）
	- `systemctl get-default`：查看当前运行级别
	- `systemctl set-default 对应运行级别` ：切换到对应运行级别
	

# 4. 文件目录命令

### 4.1 目录导航与查看

- `pwd`：查看当前工作目录绝对路径
- `ls`：列出当前工作目录的文件和目录
- `ls -l`：以列表形式列出
- `ls -a`：列出所有的，包括隐藏的（以 `.` 开头）

- `cd`：回到家目录
- `cd ~`：回到家目录
- `cd [path]`：切换到指定目录
- `cd ..`：回到工作目录上一级目录

### 4.2 目录/文件创建

- `mkdir [path]`：创建单级目录
- `mkdir -p [path]`：创建多级目录（自动创建不存在的父目录）
- `touch [file]`：创建空文件

### 4.3 目录/文件复制移动

- `cp [src] [dst]`：复制文件/目录
- `cp -r [src] [dst]`：递归复制整个目录
- `cp -rf [src] [dst]`：递归复制并强制覆盖（无需确认）
- `/cp -r [src] [dst]`：递归复制并强制覆盖（无需确认）

- `mv [src] [dst]`：移动/重命名文件/目录
- `mv -f [src] [dst]`：强制移动/重命名（⚠️ 强制覆盖，谨慎使用）

### 4.4 文件内容查看（文件查看器）

- `cat [file]`：查看小文件的完整内容
- `cat -n [file]`：查看文件内容，显示行号
- `more [file]`：分页查看文件（仅支持向下翻页）
- `less [file]`：分页查看大文件（支持上下翻页、搜索）
    - `PageDown / Space`：向下翻动一页
    - `PageUp / b`：向上翻动一页
    - `↓ / Enter`：向下翻动一行
    - `↑`：向上翻动一行
    - `g`：跳转到文件开头
    - `G`：跳转到文件末尾
    - `/[关键字]`：向下搜索关键字（n 下一个，N 上一个）
    - `?[关键字]`：向上搜索关键字
    - `q`：退出 `less` 查看

- `head [file]`：查看文件头部内容
- `tail [file]`：查看文件尾部内容
- `tail -f [file]`：实时追踪文件尾部内容（常用于日志查看）

### 4.5 目录/文件删除与属性

- `rmdir [path]`：删除指定空目录
- `rm -r [path]`：递归删除整个目录（含非空目录）
- `rm -f [path]`：强制删除文件/目录（⚠️ 谨慎使用）
- `rm -rf [path]`：强制递归删除整个目录（⚠️ 高危，谨慎使用）

- `chown [user:group] [file/dir]`：修改文件/目录所有者
- `chown -R [user:group] [path]`：递归修改目录下所有文件/目录的所有者
- `stat [file/dir]`：查看文件详细属性信息
