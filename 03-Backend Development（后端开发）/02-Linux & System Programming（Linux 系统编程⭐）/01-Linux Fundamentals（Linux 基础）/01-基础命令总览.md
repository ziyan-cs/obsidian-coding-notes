## 核心

- Shell 是 **用户与操作系统内核交互的命令行解释器**
- 命令本质是操作四大核心对象：
    - 文件系统（目录 / 文件）
    - 进程
    - 权限
    - 输入输出流

## 1. 帮助命令

- `man [命令/配置文件]`：获得外部命令或配置文件帮助信息
- `help [命令]`：获得 Shell 内置命令的帮助信息
- `[命令] -- help`：快速查看命令参数概述

## 2. 关机 & 重启

- `halt`：立即关机
- `reboot`：立即重启
- `sync`：同步内存数据到磁盘
- `poweroff`：立即关机

- `shutdown`：默认一分钟后关机
- `shutdown -h now`：立即关机
- `shutdown -h 1`：一分钟后关机（自定义时间）
- `shutdown -r now`：立即重启

## 3. 运行级别

- 传统运行级别：
	- `0`：关机
	- `1`：单用户模式【找回丢失密码】
	- `2`：多用户状态无网络服务
	- `3`：多用户状态有网络服务
	- `4`：系统未使用保留给用户
	- `5`：图形界面（默认）
	- `6`：系统重启
	- `init [级别]`：切换运行级别（例如  `init 3`）
	

- 现代 *Ubuntu* 等系统用 `systemd 目标（target）` 替代传统运行级别，`init` 命令仅做兼容：
	- `0` ↔ `poweroff.target`（关机）
	- `3` ↔ `multi-user.target`（多用户命令行，服务器常用）
	- `5` ↔ `graphical.target`（图形界面，桌面常用）
	- `6` ↔ `reboot.target`（重启）
	- `systemctl get-default`：查看当前运行级别
	- `systemctl set-default 对应运行级别` ：切换到对应运行级别
	

## 4. 文件目录命令

- `pwd`：查看当前工作目录绝对路径
- `ls`：列出当前工作目录内容
- `ls -l`：以列表形式列出当前工作目录内容
- `ls -a`：列出所有文件和目录，包括隐藏的（以 `.` 开头）
- `cd`：回到家目录
- `cd ~`：回到家目录
- `cd [目录]`：切换到指定目录
- `cd ..`：回到工作目录上一级目录

- `mkdir [目录]`：创建单级目录
- `mkdir -p /home/animal/dog`：创建多级目录（`animal` 原来没有）
- `rmdir [目录]`：只能删除指定空目录
- `rmdir -rd [目录]`：强制删除指定目录

- `cat`：查看文件内容
- `less`：分页查看大文件
- `head`：查看文件头部内容
- `tail`：查看文件尾部内容
- `cp`：复制文件 / 目录
- `mv`：移动 / 重命名文件 / 目录
- `rm`：删除文件 / 目录
- `touch`：创建空文件 / 修改文件时间

- `chmod`：修改文件 / 目录权限
- `chown`：修改文件 / 目录所有者
- `stat`：查看文件详细属性信息
