

# 0. 核心（Linux 哲学 + 常用命令）


### Linux哲学

- 一切皆文件
- 命令只实现一个功能
- 命令可组合、管道协作
- 纯文本配置
- 尽量避免图形化
- 权限严格、安全优先
- 开源、自由、可定制


### 常用命令

- `type [cmd]`： 判断命令类型
		
	- `builtin` 内置命令：
		- 本质：`Shell` 自带（当前 `Shell` 执行）
		- 代表：`cd` `help` `type` `echo` `export`
		
	- `external` 外部命令：
		- 本质：独立二进制文件（ `fork` 子进程执行）
		- 代表：`ls` `cat` `find` `date` `which`

# 2. 文件 & 目录（🔥重点）


### 路径

- `pwd`：查看当前绝对路径
- `cd`：进入家目录
- `cd ~`：进入家目录
- `cd [dir_path]`：切换至指定目录
- `cd ..`：回到上一级目录


### ls / tree

- `ls`：竖直结构 列出目录子项
	- `-l`：（列表形式）
	- `-h`：（文件大小 K / M / G ）
	- `-a`：（所有，包括 `.` ）
	- `-t`：（新修改排前）
	- `-S`：（大文件排前）
	- 总用量：文件、子目录、软链接占用磁盘块数之和

- `tree`：树形结构 展示目录结构
	- `-L [num]`：只显示 `[num]` 层深度目录结构
	- `-h`：（文件大小 K / M / G ）
	- `-a`：（所有，包括 `.` ）
	- `-d`：（只显示目录）
	- `-f`：（文件绝对路径）
	- `-I "*_name"`：（排除指定文件 / 目录）


### mkdir / touch


- `mkdir [path]`：创建单级目录
- `mkdir -p [path]`：创建多级目录
- `touch [file_name]`：创建空文件


### cp / mv / rm


- `cp [src] [dst]`：复制文件 / 目录
	- `-r`：（递归复制所有子项）
	- `-f`：（强制覆盖）

- `mv [src] [dst]`：移动 / 重命名文件 / 目录
	- `-f`：（强制覆盖）⚠️

- `rm [path]`：删除文件 / 目录
	- `-r`：（递归删除所有子项）
	- `-f`：（强制删除）
	- `-rf`：（强制递归删除所有子项）⚠️


### ln（软 / 硬链接🔥）


- `ln -s [src] [dst]`：原文件 / 目录 快捷方式
- `rm -rf [dst]`：删除软链接，目录末尾不加 `/`

- `ln [src] [dst]`：原文件 / 目录 文件备份
- `rm -rf [dst]`：删除硬链接

# 3. 文件内容操作


### cat / tac


- `cat [file_name]`：输出小文件的完整内容
- `cat -n [file_name]`：输出文件内容，并显示行号

- 输出控制台
	- `echo "内容"`：输出指定内容到控制台
    - `echo [环境变量]`：查看系统环境变量
		- `$PATH`：命令搜索路径
		- `$HOME`：当前用户家目录
		- `$USER`：登录用户名
		- `$PWD`：当前所在目录
		- `$SHELL`：当前使用的 shell
		- `$LANG`：系统语言
		- `$HOSTNAME`：主机名

- 输出重定向
	- `echo "内容" > [file_name]`
	- 覆盖文件原有内容

- 追加
	- `echo "内容" >> [file_name]`
	- 文件尾添加


### more / less


- `more [file_name]`
	- 分页查看（只支持下翻页）
	- `q`：退出

- `less [file_name]`
	- 分页查看（支持上下翻页、搜索）
    - `↑`：上一行
	- `↓ / Enter`：下一行
    - `PageUp / b`：上一页
	- `PageDown / Space`：下一页
    - `g`：跳到开头
    - `Shift + G`：跳到末尾
	- `/[关键字]`：向下搜索（n 下一个，N 上一个）
	- `?[关键字]`：向上搜索
	- `q`：退出


### head / tail（🔥日志）



- `head [file_name]`：输出头部 10 行
-  `head -n [num] [file_name]`：输出文件头部指定行数内容
	- 如 `head -n 5 [file_name]`：查看前 5 行
- `tail [file_name]`：输出尾部 10 行
- `tail -n [num] [file_name]`：输出文件尾部指定行数内容
	- 如 `tail -n 5 [file_name]`：查看后 5 行

- 实时监控
	- `tail -f [file_name]`：实时追踪文件尾部更新内容
		- 持续占用当前终端进程，无法输入新命令
		- 需打开 **多终端 / 多窗口** 执行后续操作
		- `Ctrl + C`：退出执行中的命令

# 5. 压缩与归档

### tar（重点🔥）


- **tar 指令选项说明**
    - `-c`：产生 `.tar` 打包文件
    - `-v`：显示详细信息
    - `-z`：打包同时压缩
    - `-x`：解包 .tar 文件
    - `-f`：指定压缩后的文件名（**必须放在最后**）

- **仅打包（.tar ）**
	- `tar -cvf [archive_name].tar [src]`：仅归档，不压缩
	- `tar -xvf [archive_name].tar`：解压 `.tar` 包

- **打包压缩（.tar.gz ）**
	- `tar -zcvf [archive_name].tar.gz [src]`：打包并压缩文件/目录
	- `tar -zxvf [archive_name].tar.gz`：解压 .tar.gz 包到当前目录
	- `tar -zxvf [archive_name].tar.gz -C [dst]`：解压到指定目录
	- `tar -ztvf [archive_name].tar.gz`：查看包内文件，不解压

- **高压缩率（tar.bz2 ）**
	- `tar -jcvf [archive_name].tar.bz2 [src]`：打包并高压缩文件和目录
	- `tar -jxvf [archive_name].tar.bz2`：解压 `.tar.bz2` 包

- **关键注意事项**
	- `tar` 的 `-f` 参数必须放在最后，紧跟压缩包名
	- `tar` 默认保留原文件，`gzip` 默认删除原文件
	- 跨平台优先用 `.tar.gz` 或 `.zip`
	- 归档大文件用 `.tar.bz2` 
	- 解压前先查看包内文件，避免覆盖同名文件


### gzip / bzip2

- **单文件压缩（.gz ）**
	- `gzip [file_name]`：压缩单个文件
	- `gzip -k [file_name]`：（保留原文件）
	- `gunzip [archive_name].gz`：解压 `.gz` 文件，恢复原文件

- **跨平台压缩（.zip ）**
	- `zip -r [archive_name].zip [src]`：递归压缩目录/文件
	- `unzip [archive_name].zip`：解压到当前目录
	- `unzip [archive_name].zip -d [btc]`：（到指定目录）
	- `unzip -l [archive_name].zip`：查看包内文件，不解压


# 6. 时间与系统信息


- `date`：显示当前完整时间
- `date "+%Y-%m-%d %H:%M:%S"`：显示自定义格式时间
	- `date +%Y`：完整年份（2020）
	- `date +%y`：两位年份（20）
	- `date +%m`：月（01-12）
	- `date +%d`：日（00-31）
	- `date +%H`：时（00-23）
	- `date +%M`：分（00-59）
	- `date +%S`：秒（00-59）
	- **多空格 / 多参数** 必须加引号（单 / 双都可以）
- `sudo date -s "字符串具体时间"`：设置当前时间
	- `hwclock -w`：同步网络时间（设置时间后必做）
	- `ntpdate ntp.aliyun.com`：`ntp` 同步时间（推荐）

- `cal`：显示本月日历
- `ncal`：垂直显示日历
- `cal -3`：显示 **上月** + **本月** + **下月** 三连日历
- `cal -m 3`：显示指定月份日历
- `cal -m 3 2020`：显示指定年份月份日历
- `cal -j`：显示是一年中的第几天
- `cal -w`：显示是一年中的第几周
- `cal 2020`：显示 `2020` 年所有月份日历

  
# 7. 历史命令 & 快捷键

- `history`：显示所有的历史命令
- `history 10`：显示最近使用过的 10 个指令
- `history -c`：清空当前所有历史命令（⚠️ 谨慎使用）
- `!`：执行上一条命令
- `!5`：执行历史编号为 5 的指令
- `!!`：重复执行上一条命令
