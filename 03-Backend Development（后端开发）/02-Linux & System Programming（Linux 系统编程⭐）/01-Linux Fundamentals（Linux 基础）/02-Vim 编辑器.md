
# 0. 核心

- 了解熟悉 快捷键+命令
- `vim [file]`：打开/新建文件，进入正常模式

# 1. Vim 快捷键

- `i`：进入插入模式（编写）
- `:`：进入底行命令模式（提供相关指令）
- `esc`：回退正常模式（支持移动光标，删除字符和整行）

- `yy`：复制当前行
- `p`：粘贴剪贴板内容
- `dd`：删除（剪切）当前行
- `u`：撤销上一步操作
- `Ctrl + r`：重做撤销的操作

- `gg`：跳转到文件首行
- `Shift + g`：跳转到文件尾行
- `^`：跳转到行首
- `$`：跳转到行尾

# 2. Vim 命令

- `:w`：保存文件
- `:q`：退出 Vim
- `:q!`：强制退出不保存
- `:wq`：保存并退出
