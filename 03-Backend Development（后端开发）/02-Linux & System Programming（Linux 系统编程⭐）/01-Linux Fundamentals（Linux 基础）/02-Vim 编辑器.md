

# 0. 核心

- 熟悉 快捷键+命令
- `vim [file]`：打开 / 新建文件，进入正常模式

# 1. 模式切换

- **基础模式**
	- 正常模式：默认，用于移动光标、复制删除 
	- 插入模式 ：用于编辑输入内容
	- 命令模式 ：执行保存、退出、替换等指令

- **切换**
	- `i`：进入插入模式（编写）
	- `:`：进入底行命令模式（提供相关指令）
	- `esc`：回退正常模式（支持移动光标，删除字符和整行）

# 2. 光标移动

- `^`：跳转到行首
- `$`：跳转到行尾

- `gg`：跳转到文件首行
- `Shift + g`：跳转到文件尾行

# 3. 编辑操作

- **编辑**
	- `yy`：复制当前行
	- `p`：粘贴剪贴板内容
	- `dd`：删除（剪切）当前行

- **撤销**
	- `u`：撤销上一步操作
	- `Ctrl + r`：重做撤销的操作

# 4. 查找与替换

- `:w`：保存文件
- `:q`：退出 Vim
- `:q!`：强制退出不保存（⚠️ 谨慎使用）
- `:wq`：保存并退出（`zz` 快捷键等效）

- `:w [new_file]`：另存为新文件 
- `:e [file]`：打开另一个文件
