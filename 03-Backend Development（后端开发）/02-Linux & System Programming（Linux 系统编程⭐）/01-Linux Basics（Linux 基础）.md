## 核心

- Shell 是 **用户与操作系统内核交互的命令行解释器**
- 命令本质是操作四大核心对象：
    - 文件系统（目录 / 文件）
    - 进程
    - 权限
    - 输入输出流
- 学习目标：
	- 通过命令行直接操作文件、目录、进程和系统环境
	- 建立「系统资源如何被命令操作」的整体理解
- 重点：
    - 路径与目录
    - 文件查看与编辑
    - 权限
    - 进程观察
    - 管道与重定向

## 运行级别说明

- 传统运行级别：
	- `0`：关机
	- `1`：单用户模式【找回丢失密码】
	- `2`：多用户状态无网络服务
	- `3`：多用户状态有网络服务
	- `4`：系统未使用保留给用户
	- `5`：图形界面（默认）
	- `6`：系统重启
	- `init [级别]`：切换运行级别

- 现代 *Ubuntu* 等系统用 `systemd 目标（target）` 替代传统运行级别，`init` 命令仅做兼容：
	- `0` ↔ `poweroff.target`（关机）
	- `3` ↔ `multi-user.target`（多用户命令行，服务器常用）
	- `5` ↔ `graphical.target`（图形界面，桌面常用）
	- `6` ↔ `reboot.target`（重启）
	- `systemctl get-default`：查看当前运行级别
	- `systemctl set-default 对应运行级别` ：切换到对应运行级别

##  关键命令类别

### 1. Vim 快捷键

- `vim 文件名`：打开/新建文件，进入正常模式

- `i`：进入插入模式（编写）
- `:`：进入底行命令模式（提供相关指令）
- `esc`：回退正常模式（支持移动光标，删除字符以及整行）

- `yy`：复制当前行
- `p`：粘贴剪贴板内容
- `dd`：删除（剪切）当前行
- `u`：撤销上一步操作
- `Ctrl + r`：重做撤销的操作

- `gg`：跳转到文件首行
- `G`：跳转到文件尾行
- `^`：跳转到行首
- `$`：跳转到行尾

- `:w`：保存文件
- `:q`：退出 Vim
- `:q!`：强制退出不保存
- `:wq`：保存并退出

### 2. 关机 & 重启

- `halt`：立即关机
- `reboot`：立即重启
- `sync`：同步内存数据到磁盘
- `poweroff`：立即关机

- `shutdown`：默认一分钟后关机
- `shutdown -h now`：立即关机
- `shutdown -h 1`：一分钟后关机（自定义时间）
- `shutdown -r now`：立即重启

### 3. 用户管理

- `sudo ***`：*Ubuntu* 下以 root 权限执行命令

- `useradd 用户名`：创建新用户（默认：`/home/用户名` ）
- `useradd -d /home/*** 用户名`： 指定家目录路径
- `useradd -g 用户组 -u 用户名`： 指定用户组
- `adduser 用户名`:  *Ubuntu* 强烈推荐
- `userdel 用户名`：删除用户（保留家目录，无法登录）
- `userdel -r 用户名`：删除用户（删除家目录）

- `passwd`：修改当前用户密码
- `passwd 用户名`：修改用户密码

- `groupadd 组名`：创建用户组
- `groupdel 组名` ：删除用户组
- `usermod -g 组名 用户名`：修改用户主组

- `su 用户名`：仅切换用户身份，不切换环境
- `su - 用户名`：完整切换用户身份与环境
- `id`：查看当前用户 `UID GID 组`
- `id 用户名`：查看用户 `UID GID 组`
- `exit`：退出进程
- `logout`：退回最初登录的用户

- `Whoami`：查看 **当前有效用户身份
- `Who am i`：查看 **最初登录的用户身份
- `who`：查看 **所有当前登录系统的用户**
- `w`：`who` 的增强版，额外显示用户的操作、负载等信息

### 3. 路径与目录

- `pwd`：查看当前路径
- `ls`：列出目录内容
- `cd`：切换目录
- `mkdir` / `rmdir`：创建 / 删除目录

3. 文件查看与编辑

- `cat`：查看文件内容
- `less`：分页查看大文件
- `head`：查看文件头部内容
- `tail`：查看文件尾部内容
- `cp`：复制文件 / 目录
- `mv`：移动 / 重命名文件 / 目录
- `rm`：删除文件 / 目录
- `touch`：创建空文件 / 修改文件时间

4. 权限与属性

- `chmod`：修改文件 / 目录权限
- `chown`：修改文件 / 目录所有者
- `stat`：查看文件详细属性信息

1. 进程与系统观察

- `ps`：查看进程状态
- `top / htop`：实时监控系统进程 / 资源
- `kill`：终止指定进程

1. 重定向与管道

- `>、>>、<`：输出 / 输入重定向
- `|`：管道符，连接命令输出与输入
