



# 1. 帮助系统

- `which [cmd]`：查看命令的 `PATH` 指定目录

- `type [cmd]`： 判断命令类型
		
	- `builtin` 内置命令：
		- 本质：`Shell` 自带（对当前 `Shell` 执行）
		- 代表：`cd` `help` `type` `echo` `export`
		
	- `external` 外部命令：
		- 本质：独立二进制文件（在 `fork` 子进程执行）
		- 代表：`ls` `cat` `find` `date` `which`

- 查看帮助信息
		
	- `help [..]`
		- 内置命令
		
	- `[..]--help`
		- 外部命令
		
	- `man [..]`
		- 外部命令 / 配置文件 （所有）
		- 纯文本、快速查参数
		
	- `info [..]`（ `man` 的进阶补充）
		- 外部命令 / 工具  （仅 **GNU** 开发的）
		- 结构化、超详细

# 2. 文件 & 目录（🔥）

### 路径（绝对 / 相对）

- `pwd`：查看当前绝对路径
- `cd`：进入家目录
- `cd ~`：进入家目录
- `cd [dir_path]`：切换至指定目录
- `cd ..`：回到上一级目录

### ls / tree

- `ls`：读取目录 **inode**， 列出目录子项
	- `-l`：（列表）
	- `-h`：（ K / M / G ）
	- `-a`：（所有）
	- `-t`：（新修改在前）
	- `-S`：（大文件在前）
	- `-R [num]`：（子目录深度，默认 1 ）

- `tree`：树形结构，展示目录结构
	- `-f`：（文件绝对路径）
	- `-h`：（ K / M / G ）
	- `-a`：（所有）
	- `-d`：（只目录）
	- `-L [num]`：（子目录深度，默认 1）
	- `-I "***"`：（排除指定）

### mkdir / touch

- `mkdir [dir_path]`：创建单级目录
	- `-p`：（多级目录）
- `touch [file_name]`：创建空文件

### cp / mv / rm

- `cp [src] [dst]`：复制文件 / 目录
	- `-r`：（递归所有）
	- `-f`：（强制覆盖）⚠️

- `mv [src] [dst]`：移动 / 重命名文件 / 目录
	- `-r`：（递归所有）
	- `-f`：（强制覆盖）⚠️

- `rm [dir_path/file_name]`：删除文件 / 目录
	- `-r`：（递归所有）
	- `-f`：（强制删除）⚠️

### ln（软 / 硬链接🔥）

- `ln -s [src] [dst]`：原文件 / 目录 快捷方式
- `rm -rf [dst]`：删除软链接，目录末尾不加 `/`

- `ln [src] [dst]`：原文件 / 目录 文件备份
- `rm -rf [dst]`：删除硬链接

	- `grep -r [关键字] [path]`：

### 管道 |

- `|`：前输出作为后输入，只传正常输出

# 5. 压缩与解压

### tar（🔥）

- 选项
    - `-c`：产生 `.tar` 打包文件
    - `-v`：显示详细信息
    - `-z`：打包并压缩
    - `-x`：解包 **.tar** 文件
    - `-f`：指定压缩后的文件名（必须放在最后）

- 仅打包（**.tar** ）
	- `tar -cvf [archive_name].tar [src]`：归档不压缩
	- `tar -xvf [archive_name].tar`：解压到当前目录

- 打包且压缩（**.tar.gz** ）
	- `tar -zcvf [archive_name].tar.gz [src]`：归档压缩
	- `tar -zxvf [archive_name].tar.gz`：解压到当前目录
	- `tar -zxvf [archive_name].tar.gz -C [dst]`：解压到指定目录
	- `tar -ztvf [archive_name].tar.gz`：查看包内文件，不解压

- 高压缩率（**tar.bz2** ）
	- `tar -jcvf [archive_name].tar.bz2 [src]`：归档高压缩
	- `tar -jxvf [archive_name].tar.bz2`：解压到当前目录

- 注意
	- `tar` 的 `-f` 参数必须放在最后，紧跟压缩包名
	- `tar` 默认保留原文件，`gzip` 默认删除原文件
	- 跨平台优先用 `.tar.gz` 或 `.zip`
	- 归档大文件用 `.tar.bz2` 
	- 解压前先查看包内文件，避免覆盖同名文件

### gzip / bzip2

- 单文件压缩（.gz ）
	- `gzip [file_name]`：压缩单个文件
	- `gzip -k [file_name]`：（保留原文件）
	- `gunzip [archive_name].gz`：解压到当前目录

- 跨平台压缩（.zip ）
	- `zip -r [archive_name].zip [src]`：递归压缩目录/文件
	- `unzip [archive_name].zip`：解压到当前目录
	- `unzip [archive_name].zip -d [btc]`：（到指定目录）
	- `unzip -l [archive_name].zip`：只查看


# 6. 系统信息

### date

- `date`：显示当前完整时间
- `date "+%Y-%m-%d %H:%M:%S"`：显示自定义格式时间
	- 多空格 / 多参数 必须加引号
- `sudo date -s "字符串具体时间"`：设置当前时间
- `hwclock -w`：同步网络时间
- `ntpdate ntp.aliyun.com`

### uptime

- `uptime`：显示系统运行时间、当前用户数、负载信息 
- `当前时间 运行时长 用户数 平均负载(1分钟/5分钟/15分钟)`


### infconfig / ip addr

- `ifconfig`：查看所有网卡信息
- `ifconfig [网卡名] down/up`：关闭 / 启用网卡

- `ip addr`：查看所有网卡的IP地址、状态信息
- `ip link`：查看网卡物理启用状态
