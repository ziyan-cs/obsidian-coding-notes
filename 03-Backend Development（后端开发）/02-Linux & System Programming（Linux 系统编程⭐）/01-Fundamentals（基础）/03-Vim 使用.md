

# 1. 模式切换



- 基础模式
	- 正常模式：移动光标、复制删除 
	- 插入模式：编辑输入内容
	- 命令模式：保存、退出、替换等

- 切换
	- `i`：进入插入模式
	- `:`：进入底行命令模式
	- `esc`：回退正常模式

# 2. 基础操作

### 光标移动

- 单词级
	- `w`：跳到下个单词开头
	- `b`：跳到上个单词开头

- 行级
	- `0`：跳转到行首
	- `^`：跳转到行首
	- `$`：跳转到行尾

- 文件级
	- `gg`：跳到文件首行
	- `Shift + G`：跳到文件尾行
	- `Ctrl + U`（上翻半页）
	- `Ctrl + D`（下翻半页）

### 打开 / 保存 / 退出

- `vim [file_name]`：打开进入正常模式

- 在命令模式
	- `:w`：保存
	- `:q`：退出
	- `:q!`：强制退出不保存
	- `:wq`：保存并退出（ `zz` 快捷键）
	- `:w [file_name]`：另存为新文件 
	- `:e [file_name]`：打开另一个文件

- 编辑模式不能用 `Ctrl+s` ，会导致终端卡死
	- 按 `Ctrl+q` 恢复

### 删除 / 复制 / 粘贴

- `dd`：剪切当前整行
- `d$`：剪切从光标到行尾
- `yy`：复制当前整行
- `p`：粘贴到光标后
- `u`：撤销上一步操作
- `Ctrl + R`：恢复撤销

# 3. 查找与替换（🔥）

### 查找文本

- `/[关键词]`：向下查找
- `n`：跳转下一个匹配项
- `Shift + N`：跳转上一个匹配项

### 全局替换

- `:%s/old/new/g`：替换所有
- `:s/old/new/g`：替换当前行
- `:%s/old/new/gc`：每个替换并确认
