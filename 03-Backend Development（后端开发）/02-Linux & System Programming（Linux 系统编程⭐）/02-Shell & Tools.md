### 什么是 Shell（bash/zsh）

- `Shell` 是 **用户与操作系统内核交互的命令行解释器**

- 命令本质是操作四大核心对象：
    - 文件系统（目录 / 文件）
    - 进程
    - 权限
    - 输入输出流
