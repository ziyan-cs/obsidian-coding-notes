

# 0. 核心

 **VSCode** + **CMake** 插件：

# 1. 初始化配置

- 编写 CMakeLists.txt 配置文件
- **Ctrl + Shift + P -> CMake: 配置**
	- 读取配置文件，在 `build` 生成构建文件

# 2. 日常开发操作

- ⚙**生成 (Build)** ：
    - 编译最新修改的代码
    - 更新 `build` 目录下的可执行文件
    - 生成 `文件名/项目名/main.exe` 可执行文件到`build` 目录

- 🐞**调试 (Debug)** ：
    - 自动加载调试配置
    - 程序运行并暂停在断点处
    - 左侧面板显示变量值

- ▶️**运行 (Run)** ：
    - 直接执行程序，查看输出结果
    - 不进入调试模式，速度快

# 3. 调试面板

- **Ctrl + Shift + P -> 开始调试**，进入调试模式

| 操作       | 简写 / 按钮  | 作用                |
| :------- | :------- | :---------------- |
| **断点**   | 🔴       | toggles 断点        |
| **继续**   | F5 / ▶️  | 程序运行到下一个断点或结束     |
| **单步跳过** | F10 / ⏭️ | 执行一行代码，不进入函数内部    |
| **单步进入** | F11 / ↓  | 执行一行代码，进入函数内部查看   |
| **查看变量** | 左侧面板     | 自动监控当前作用域变量，完全可视化 |
| **修改变量** | 双击值      | 直接修改变量值，实时生效      |
